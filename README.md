# AHTECH Static Website

This repository has been migrated from Laravel/MySQL to a fully static Nuxt 4 website.

## Stack

- Nuxt 4
- Nuxt I18n
- Nuxt UI
- Tailwind CSS through Nuxt UI

## What is included

- Fully static public marketing site
- English and Vietnamese routes via Nuxt I18n
- Service detail pages
- Static blog listing and article detail pages
- Partner logo strip sourced from `public/partner_logos`
- Static contact page with email-draft CTA

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## Notes

- The old PHP/Laravel source has been removed from this repository.
- The old Supabase-backed dynamic layer has also been removed.
- The current build succeeds with `npm run build`.
- Localized routes are generated by Nuxt I18n with English as the default locale and Vietnamese under `/vi`.
